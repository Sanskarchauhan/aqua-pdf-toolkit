
This directory contains PDF.js Express library files.
To use the PDF viewer functionality, please download the PDF.js Express library files from their official website 
and place them in this directory. The application will look for these files at runtime.

Required files include:
- webviewer.min.js
- core/
- ui/
- pdf/

Note: When downloading PDF.js Express, make sure to extract the contents of the 'lib' folder here.
